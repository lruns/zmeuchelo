# ЗМЕЮЧЕЛО

Маленькая браузерная змейка по мотивам одного скриншота из тг Юлика.
Вся графика — в наивной Paint-эстетике: бледно-жёлтое поле, фигуры из
эллипсов и прямоугольников с чёрной обводкой и видимыми линиями пересечений.

Один самодостаточный файл `index.html` — открывается прямо в браузере
(и на телефоне, и с `file://`). Никаких сборок и зависимостей.

Игра сгенерирована одним запросом. Ладно, двумя. (ну ладно, 5-6, кто считает.)

## Как играть

- Стрелки / WASD / свайп — поворот.
- Собирай красные яблоки-рты, расти.
- Ёлка и пруд — смертельны (об этом честно предупреждают таблички).
  Мужики — безобидны: они просто воруют яблоки и комментируют.
- Врезаться в собственный хвост тоже нельзя.
- Пауза — кнопка в углу или клавиша Esc / P. Там же тумблеры звука и музыки.
